====================
== Cocktail Demos ==
====================

Updated on 2012-08-16

This file gathers the description and the status of each demo.

clock
-----

Description: an analog clock.
Tested targets: JavaScript & Flash
Online demo: http://demos.silexlabs.org/cocktail/clock/
Status: source code obsolete. Uses cocktail's obsolete DOM abstraction API.


cocktailGallery
---------------

Description: a simple gallery with thumbnails.
Tested targets: JavaScript & Flash
Online demo: none
Status: source code obsolete. Uses cocktail's obsolete DOM abstraction API.


simple-webapp
-------------

Description: a simple webapp aiming to reproduce an iPhone menu screen, with each menu icon pointing to text & pictures lists or to web links.
Tested targets: JavaScript & Flash
Online demo: http://demos.silexlabs.org/cocktail/simple-webapp/
Status: source code obsolete. Uses cocktail's obsolete DOM abstraction API.


ski game
--------

Description: this is the classic little ski game of GM2D, which was ported to Cocktail.
	=> http://gm2d.com/2009/02/simple-flash-game-in-haxe/
Tested targets: JavaScript & Flash
Online demo: http://demos.silexlabs.org/cocktail/ski/
Status: source code obsolete. Uses cocktail's obsolete DOM abstraction API.
